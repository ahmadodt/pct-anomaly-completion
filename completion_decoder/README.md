# Completion Decoder Experiment

This folder contains a point-cloud completion experiment on ModelNet10. It trains a small PCT-style encoder/decoder to reconstruct a missing local region from a partial point cloud.

The current task is:

1. sample a full point cloud from a ModelNet10 `.off` mesh
2. remove a local nearest-neighbor region to create the missing shape
3. feed the remaining partial point cloud into the model
4. predict the missing points
5. train with symmetric Chamfer distance between predicted and real missing points

## Environment

Use the repository Python 3.11 environment:

```powershell
..\.venv311\Scripts\Activate.ps1
python -m pip install -r requirements.txt
```

## Data

The loader expects ModelNet10 in this layout:

```text
completion_decoder\
  data\
    ModelNet10\
      bathtub\
        train\
        test\
      ...
```

If the folder is missing, `data.py` attempts to download and extract:

```text
http://3dvision.princeton.edu/projects/2014/3DShapeNets/ModelNet10.zip
```

If automatic download fails, download/extract it manually and either place it at `completion_decoder\data\ModelNet10` or set:

```powershell
$env:MODELNET10_ROOT="C:\path\to\ModelNet10"
```

## Run

From this folder:

```powershell
..\.venv311\Scripts\Activate.ps1
python main.py --epochs 1 --batch_size 2 --test_batch_size 2 --num_workers 0 --num_points 128 --num_missing_points 32
```

Normal training can use larger settings:

```powershell
python main.py --epochs 100 --batch_size 32 --test_batch_size 16 --num_points 512 --num_missing_points 128
```

The best checkpoint is saved to:

```text
checkpoints\<exp_name>\models\model.pct
```


## Caveats

- This is a compact research prototype, not a paper-quality completion model.
- The missing region is synthetically generated by removing nearest neighbors around a random point.
- Chamfer loss measures geometric closeness but does not enforce point order.
